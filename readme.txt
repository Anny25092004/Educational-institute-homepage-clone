Educational Institute Website – Homepage Clone (Frontend)
 Overview

This project is a frontend-only homepage clone of an educational institute website, created solely for learning and portfolio purposes.

The objective of this project was to understand and implement real-world responsive layouts, mobile navigation, and clean UI structuring using core web technologies.

This is not the complete website — only the homepage UI has been recreated.

 Key Features

Responsive homepage layout (desktop, tablet, mobile)

Mobile-first navigation with hamburger menu

Toggle-based mobile menu using Vanilla JavaScript

Responsive grid-based content sections

Fixed header with proper layout handling

Clean and readable UI structure

 Technologies Used

HTML5 – Semantic structure

CSS3 – Flexbox, Grid, Media Queries

Vanilla JavaScript – Navigation toggle logic

 Responsiveness & Testing

Tested on:

Desktop browsers

Real mobile devices

Ensured:

No horizontal scrolling

Stable layout across screen sizes

Usable mobile navigation

How to Run the Project

Clone or download the repository

Open index.html in any modern browser
OR

Use Live Server in VS Code

Project Purpose

This project was built to:

Practice responsive frontend development

Understand real-world UI behavior

Serve as a portfolio demonstration project

No backend functionality or dynamic data is included.
